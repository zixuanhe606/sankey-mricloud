# MRICloud Sankey Diagram

This repository hosts the interactive Sankey diagram for Homework 5.

The diagram displays MRICloud brain parcellation data for Type 1 structures, starting from intracranial volume (ICV) and showing multiple hierarchy levels.

## Live Webpage

The hosted Sankey diagram is available here:

https://zixuanhe606.github.io/sankey-mricloud/sankey.html

## File Included

- `sankey.html`: Interactive Plotly Sankey diagram exported as an HTML file.

## Description

The Sankey diagram visualizes the hierarchical structure of MRICloud brain regions:

- Intracranial volume (ICV)
- Level 1 major brain regions
- Level 2 sub-regions
- Level 3 lobes and sub-structures
- Level 4 gyri-level structures

The plot was created using Python and Plotly.
